# Web Ip
Get any website's ip adress easily!
## Installation
```sh
git clone https://github.com/notkar1m/webip.git
```
## Usage
```sh
python3 webip google.com
```
```sh
python3 webip websiteName
```
## Python Usage
```python
import webip
print(webip.main("google.com"))
```
